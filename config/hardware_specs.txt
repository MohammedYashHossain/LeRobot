# SO-101 Robot Arms Hardware Specifications
# Generated: August 5, 2025

## HARDWARE REQUIREMENTS

### Robot Arms
- **Model**: SO-101 (Leader and Follower)
- **Manufacturer**: Custom/DIY build
- **Degrees of Freedom**: 6 DOF (5 joints + gripper)
- **Material**: 3D printed components with metal joints

### Motors
- **Type**: Feetech Smart Servos
- **Model**: STS3215 (internal model number 777)
- **Quantity**: 12 total (6 per arm)
- **Power**: 12V DC
- **Communication**: Serial (RS485)
- **Control**: Position control with feedback

### Motor Configuration
#### Leader Arm (COM3)
1. **Shoulder Pan** - ID 1, sts3215, Degrees mode
2. **Shoulder Lift** - ID 2, sts3215, Degrees mode
3. **Elbow Flex** - ID 3, sts3215, Degrees mode
4. **Wrist Flex** - ID 4, sts3215, Degrees mode
5. **Wrist Roll** - ID 5, sts3215, Degrees mode
6. **Gripper** - ID 6, sts3215, Range 0-100 mode

#### Follower Arm (COM4)
1. **Shoulder Pan** - ID 1, sts3215, Degrees mode
2. **Shoulder Lift** - ID 2, sts3215, Degrees mode
3. **Elbow Flex** - ID 3, sts3215, Degrees mode
4. **Wrist Flex** - ID 4, sts3215, Degrees mode
5. **Wrist Roll** - ID 5, sts3215, Degrees mode
6. **Gripper** - ID 6, sts3215, Range 0-100 mode

### Communication Hardware
- **USB to Serial Converters**: 2x (one per arm)
- **Cables**: USB-C to USB-A
- **Ports**: COM3 (Leader), COM4 (Follower)
- **Baudrate**: 1,000,000 bps
- **Protocol**: Feetech serial communication
